# taobao_auto_buy

# Requirements
* python3.8
  * selenium==3.141.0
  
# How to use
1. clone project
```
git@github.com:Jesseslco/taobao_auto_buy.git
```
2. install python3.8
3. install requirements
```
pip3 install selenium==3.141.0
```
4. download chromedriver
5. configure config.json
6. run
```
python3 taobao.py
```
