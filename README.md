# taobao_auto_buy
# 淘宝自动购买

## Requirements
## 搭建环境
* python3.8
  * selenium==3.141.0
* chrome
  * chromedriver
  
## How to use
## 使用说明
1. clone project
```
git@github.com:Jesseslco/taobao_auto_buy.git
```
2. install python3.8
3. install requirements
```
pip3 install selenium==3.141.0
```
4. download chrome and chromedriver
5. configure config.json
6. run
```
python3 taobao.py
```
